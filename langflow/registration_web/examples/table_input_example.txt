생산 데이터셋을 등록하고 싶다.
dataset_key는 production으로 사용한다.
표시명은 생산 실적이다.
설명은 일자, 공정, 제품 조건, 라인별 생산 실적 데이터이다.

source_type은 oracle이다.
tool_name은 get_production_data로 사용한다.
Oracle DB key는 MES이고 테이블명은 PROD_TABLE이다.

필수 파라미터는 date이다.
format_params도 date이다.
date는 retriever tool 함수 안에서 YYYYMMDD 형식으로 변환된다.

질문 키워드는 생산, 생산 실적, 생산량, 실적, output, qty, production을 포함한다.

질문 예시는 다음과 같다.
- 2026-04-22 생산 실적 보여줘
- 어제 공정별 생산량 알려줘
- 오늘 라인별 생산 실적을 보여줘

컬럼:
- WORK_DT: date, 작업일자, YYYYMMDD
- OPER_NAME: string, 공정명
- MODE: string, 제품 mode
- DEN: string, density
- TECH: string, technology
- MCP_NO: string, MCP 번호
- PKG_TYPE1: string, 패키지 유형 1
- PKG_TYPE2: string, 패키지 유형 2
- LINE: string, 라인
- production: number, 생산 수량

주의:
- Table Catalog에는 SQL을 저장하지 않는다.
- 실제 SELECT/WITH 쿼리는 get_production_data 같은 retriever tool 함수 내부에서 작성한다.
- SQL이 필요한 경우 이 입력창에는 참고용 컬럼 설명만 적고, 실제 쿼리는 코드의 tool 함수에서 수정한다.
