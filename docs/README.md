## 문서별 용도

- `00_START_HERE`
  - 프로젝트를 처음 보는 사람을 위한 입문 문서
- `01_LANGFLOW_CONCEPT`
  - Langflow와 LangGraph를 이 프로젝트 관점에서 설명
- `02_PROJECT_STRUCTURE`
  - 디렉터리와 파일 역할 설명
- `03_BUILD_FLOW`
  - 기능을 어떤 순서로 구현하면 좋은지 설명
- `04_FUNCTION_GUIDE`
  - 핵심 함수와 진입점 정리
- `05_CUSTOM_NODE_WRITING_GUIDE`
  - 커스텀 노드를 어떤 방식으로 작성해야 하는지 설명
- `06_BRANCH_VISIBLE_LANGFLOW_FLOW`
  - 분기 가시형 Langflow 구조 설명
- `07_LANGFLOW_CANVAS_SETUP`
  - 실제 Langflow 캔버스 배치와 포트 연결 방법
